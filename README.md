## aalto cafeterias telegram bot

#### Running the bot
Clone the repo:
```bash
$ git clone https://github.com/javierhonduco/aalto-appetite && cd aalto-appetite
```
Install the dependencies
```bash
$ bundle install
```
Run it:
``bash
$ APPETITE_TELEGRAM_TOKEN=<your_telegram_token> bundle exec ruby bot.rb
```
